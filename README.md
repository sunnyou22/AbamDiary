# 앱 소개
> 💡 캘린더의 날짜를 클릭하면 그 날의 일기내용을 확인할 수 있고 응원메세지를 작성하면 메인화면에서 랜덤으로 보여줍니다  
일기 작성 횟수를 수치화하여 애니메이션으로 표현하였으며 매달 자신이 아침형인지 저녁형인지 확인할 수 있습니다 :)
